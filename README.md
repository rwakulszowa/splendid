# Splendid

Easy visualization of any data.

In short - the project aims to visualize arbitrary JSON data. It should process
the input, try to detect patterns (e.g. the fact that objects form some kinda
tree) and show a visualization.
If there's many matching interpretations of the data - the tool should simply
pick the top few and render them side by side.
In other words - whatever it takes to make sense of the data, without doing
any manual work.
